PokeReader is a 3gx plugin that replaces PCalc. It has more features for RNG purposes.

## Tools

- [A 3DS with CFW (Custom Firmware)](https://3ds.hacks.guide/)
- [Luma version 13.0.0 or higher](https://github.com/LumaTeam/Luma3DS/releases)

## Install PokeReader on 3DS

1. Download the [latest release of PokeReader](https://github.com/zaksabeast/PokeReader/releases/latest).
2. Rename `pokereader.3gx` to `default.3gx`.
3. Copy `default.3gx` to `/luma/plugins/` on your SD card. Create the directory if it does not exist.
4. Open Rosalina (`L` + `Down` + `Select`) and enable the `Plugin Loader`.

## Install PokeReader on Azahar/Lime3DS/Citra

1. Download the [latest release of PokeReader](https://github.com/zaksabeast/PokeReader/releases/latest).
2. Rename `pokereader.3gx` to `default.3gx`.
3. Open the emulator's preferences.
4. Navigate to `System` settings.
5. Check the `Enable 3GX Plugin Loader` box.
6. Go to the `Storage` tab.
7. Click `Open` next to `SDMC Directory`.
8. Copy `default.3gx` to `/luma/plugins/` in the directory. Create the directory if it does not exist.
